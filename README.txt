=====
Home Automation X10
=====

Home Automation X10(hautomationx10) is a mochad wrapper which makes it easier to issue
some commands to the underlying mochad layer.

Quick start
-----------

1. Make the USB cm15a device is plugged in a server

2. Make sure mochad is running on that server

3. open hautomation/x10/settings.py and set MOCHAD_HOST and MOCHAD_PORT to proper values

4. Start issuing commands.
